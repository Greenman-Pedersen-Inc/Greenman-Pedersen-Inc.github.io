SettingsMenu/widgets/CrashClusterSelect

Allows the user to select crash clusters by drawing a polygon around crash clusters on the map. This button is enabled once the user zooms to the cluster level. This is enabled from the CrashMap function 'refreshHandler'. The drawing event handler is in the CrashMap class and is called using 'map.on('draw.create', function(e) {...}'
